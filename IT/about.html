About This Website

This website is a project developed to demonstrate web development fundamentals including HTML, CSS, and JavaScript. It is structured with usability and design principles in mind, following a consistent layout, color scheme, and navigation system across all pages.

It includes an image gallery, a contact form with JavaScript validation, a multi-row and column table, and multiple types of navigation menus for accessibility and ease of use. All styling and scripts are written internally to maintain compliance with the project’s requirements.

The purpose is to apply theoretical knowledge into practice by building a clean, navigable, responsive website that meets educational and technical standards.
